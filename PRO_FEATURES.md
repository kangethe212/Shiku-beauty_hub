## üöÄ HOW TO USE NEW FEATURES

### Setup Videos

1. Go to Admin ‚Üí Videos
2. Click "Add Video"
3. Fill in:
   - Title and description
   - Upload video file
   - Upload thumbnail image
   - Select category
   - Add duration
   - Mark as featured (for home page)
4. Save!

Your video will appear on `/videos/` page and home page if featured.

---

### Setup Daily Offers

1. Go to Admin ‚Üí Daily Offers
2. Click "Add Daily Offer"
3. Fill in:
   - Offer title
   - Description
   - Upload image
   - Set discount percentage (e.g., 20 for 20% off)
   - Enter original price
   - Set start and end dates
   - Add terms & conditions
   - Mark as featured
4. Save!

Discounted price is calculated automatically!

---

### Setup International Pricing

1. **Add Currencies**:
   - Go to Admin ‚Üí Currencies
   - Add USD, EUR, GBP
   - Set exchange rates
   - Mark one as base currency

2. **Add Product Prices**:
   - Go to Admin ‚Üí Products
   - Enter base price
   - Optionally add USD, EUR, GBP prices
   - Save!

---

### Manage Inventory

1. Go to Admin ‚Üí Products
2. For each product, set:
   - Stock quantity
   - Low stock threshold (e.g., 5)
   - SKU (optional)
   - Weight (optional)
3. Admin shows color-coded status:
   - Green = In Stock
   - Orange = Low Stock
   - Red = Out of Stock

---

### Enable Reviews

1. **Approve Reviews**:
   - Go to Admin ‚Üí Product Reviews
   - Select reviews
   - Click "Approve selected reviews"

2. **Display on Products**:
   - Reviews appear on product pages
   - Average rating displays automatically
   - Star rating shows visually

---

### Update Social Links

1. Go to Admin ‚Üí Social Media Links
2. Add your platforms:
   - Select platform
   - Enter URL
   - Add username
   - Set display order
   - Mark as active
3. Links appear in footer and contact page

---

### Update Business Info

1. Go to Admin ‚Üí Business Information
2. Update:
   - Contact details
   - Business hours
   - About text, mission, vision
   - Upload logo
3. Save - updates appear site-wide!
